## 🛠️ Technology Stack
- **Structure**: HTML5
- **Styling**: Custom CSS3 (Dark Mode, Glassmorphism, animations)
- **Logic**: Vanilla JavaScript (ES6+)
- **Icons**: Font Awesome 6
- **Typography**: Google Fonts (Outfit & Inter)
- **Form Handling**: Formspree API (AJAX integrations)

---

## ✨ Features
1. **Interactive Neural Network Particles**: An interactive HTML5 Canvas background that renders dynamic particle connections that react and repel based on your mouse cursor movements.
2. **Dynamic Typing Effect**: Cycles through key specialties (AI/ML Solutions, Computer Vision, Deep Learning, NLP, RAG Systems) in the hero section.
3. **Responsive Timeline**: Shows my education and internship history cleanly across desktop, tablet, and mobile screens.
4. **Interactive Tabs**: Toggle seamlessly between my internships (Capabl & Skillcraft Technology).
5. **Project Category Filter**: Instantly filter my work (All, AI & ML, Computer Vision, Web & Backend, Cyber Security) via active tag selection.
6. **Integrated Contact Form**: Uses asynchronous `fetch` requests with Formspree to send messages directly from the page without redirects, complete with validation state feedback.
7. **Direct Resume Viewing**: Direct viewing and downloading of my latest resume PDF.
